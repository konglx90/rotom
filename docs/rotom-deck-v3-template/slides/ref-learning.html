3 · Hermes MoA · 三个臭皮匠讨论模式

Mixture of Agents:1 个问题 → 3 个 Agent 同时独立作答 → 第 4 个 Agent 做评委汇总 → 输出最优解。

  • 并发:3 个 worker 同时跑,不出第 4 个之前不汇总
  • 分工:不是串行独干,是真·多视角
  • 评委:汇总时再做去重 + 加权 + 排版
  • 适合:架构选型 / 方案对比 / 关键决策
