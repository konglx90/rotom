Dashboard 组件一览 / 6 大块

rotom Dashboard 不止聊天 —— 所有能力都装进同一个群下面。讲清楚这 6 块,后面所有场景都顺。

群聊

协作入口。一切挂群里,@ 谁谁接活,#reply 触发桥接。

→ 见第 3 页 rotom 30 秒 + 资产排查 3 页

Issue

任务单位。写盘必挂 issue,可抢单 / 审批 / 中断 / 续跑。

→ 见第 11-14 页 · 治理第 9 页

Memory

长期记忆。Agent 干完活自动写一条,人 review 后 recall。

→ 见第 5-6 页 持续学习

Schedule

定时任务。30s tick 扫 · interval / once 两种,派 issue 或 message。

→ 见第 7-8 页 巡检(基于 schedule)

Artifacts

产物管理。群产物 + diff 预览 + 跨群可见,代码 / 报告都落这。

→ 见 docs/INSTALL.md · /api/artifacts

Skill

可复用流程。同一类问题第 3 次出现 → Agent 提议 skill create。

→ 见第 5-6 页 持续学习 · 复用图谱
