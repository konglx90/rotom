# 创建 Issue 弹窗 UI 优化视觉对比

## 改进前后对比

### 1. 弹窗整体视觉

**改进前：**
- 普通黑色半透明遮罩（opacity: 0.5）
- 内容区内边距：24px
- 圆角：8px
- 阴影：普通大阴影
- 标题：默认字号和粗细

**改进后：** ✅
- 加深遮罩（opacity: 0.6）+ 毛玻璃效果（backdrop-filter: blur）
- 内容区内边距增大至：32px
- 圆角增大至：12px
- 阴影加深至：超大阴影（shadow-xl）
- 标题增大至：20px，加粗（font-weight: 700）
- 添加淡入+上滑动画效果

### 2. 表单输入框样式

**改进前：**
- padding: 10px
- border-radius: 4px
- focus: 2px 蓝色阴影

**改进后：** ✅
- padding 增大至：12px 14px
- 保持 4px 圆角
- focus: 3px 蓝色阴影 + 上浮1px动画
- 添加0.2s平滑过渡效果

### 3. 文本域样式（新增）

**改进前：** 使用无效样式类

**改进后：** ✅
- 专用 `formTextarea` 样式类
- min-height: 120px，支持垂直拉伸
- line-height: 1.5，提升可读性
- 获得与输入框相同的 focus 动画效果

### 4. 选择框样式（新增）

**改进前：** 默认浏览器样式

**改进后：** ✅
- 自定义下拉箭头图标（SVG）
- appearance: none，完全自定义
- cursor: pointer，明确可点击
- 获得与输入框相同的 focus 动画效果

### 5. 复选框样式

**改进前：**
- 行内样式，对齐 center
- 无背景色，普通间距
- 标签字体小（12px）

**改进后：** ✅
- 背景色：--color-background
- 内边距：8px 10px
- 圆角：4px
- 对齐：flex-start（顶对齐）
- 标签字体增大至：13px
- `/plan` 标签高亮显示（蓝色+粗体）
- 说明文字行高优化（1.4）

### 6. 按钮样式

**改进前：**
- 保存按钮：普通蓝色，无阴影
- 取消按钮：无边框悬停效果
- 按钮过渡：无平滑动画

**改进后：** ✅
- 保存按钮：添加蓝色阴影，悬停时加深颜色+上浮1px+阴影增大
- 取消按钮：悬停时边框变蓝+文字变蓝+上浮1px
- 所有按钮：添加0.2s平滑过渡动画
- 操作区添加顶部边框分隔线
- 按钮padding和字号增大

### 7. 关闭按钮样式

**改进前：**
- 普通悬停效果
- 固定大小

**改进后：** ✅
- 悬停时旋转90度动画
- 增大点击区域（32px×32px）
- 添加背景色过渡动画

## CSS 变量增强

新增了以下 CSS 变量完善设计系统：

```css
--color-text-primary: #0F172A;
--color-text-secondary: #64748B;
--text-2: #64748B;              /* 辅助文本色 */
--font-mono: 'Fira Code', ...;  /* 等宽字体 */
```

## 动画效果清单

所有动画均为 0.2s ease 过渡：

1. **弹窗出现**：fadeIn（透明度 0→1） + slideUp（上移20px→0）
2. **输入框 focus**：边框变色 + 3px阴影 + 上浮1px
3. **按钮悬停**：颜色变化 + 上浮1px + 阴影增大
4. **关闭按钮悬停**：旋转90度
5. **取消按钮悬停**：边框和文字变色

## 设计系统一致性

- ✅ 使用 `--color-*` 系列颜色变量
- ✅ 使用 `--radius-*` 系列圆角变量
- ✅ 使用 `--spacing-*` 系列间距变量
- ✅ 使用 `--shadow-*` 系列阴影变量
- ✅ 添加了缺失的组件样式变量

## 性能影响

- ✅ 纯 CSS 实现，无额外 JavaScript
- ✅ 使用 CSS 变量，便于主题切换
- ✅ 动画使用 transform 和 opacity，GPU 加速
- ✅ 无额外图片资源（使用内联 SVG）

## 浏览器兼容性

- ✅ backdrop-filter：现代浏览器支持（降级方案：仅透明度）
- ✅ appearance: none：现代浏览器支持
- ✅ CSS 变量：现代浏览器支持
- ✅ transition 动画：主流浏览器支持
